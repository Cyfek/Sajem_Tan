Normal Text to Reformed Orthography Converter

This is a simple AutoHotkey program that converts whatever you type into the Sajem Tan Reformed Orthography. It will remap the Original Orthography's digraphs to the Reformed Orthography's diacritic-marked characters. In order to run this, you need to have AutoHotkey installed on your computer. Follow the download instructions on the website, download this program from the link below, and just double tap the script. To toggle the conversion on and off, use the [ScrollLock] key.
